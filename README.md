# Traintorch (Alpha Release)
 Package for live visualization of metrics during training of a machine learning model
 
 # Installation
 
 ```
 pip install traintorch
  ```

 ```
 pip install git+https://github.com/rouzbeh-afrasiabi/traintorch.git
 ```
